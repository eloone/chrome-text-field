Update 0.0.0.5

26th June 2013

In this new release chrome sync is made available.

If you activate syncing in chrome, the text that you save in Text Field will sync across your different machines where you enabled chrome sync. The rest of the features remain the same.

If you don't activate chrome sync, your data will be saved only locally just as before.

For any feedback visit https://github.com/eloone/chrome-text-field or use the feedback link on chrome webstore.
